Github Statistics - Add Token

This will grant you 5000 requests per hour instead of the default 60 requests per hour when unauthenticated.
To authenticate you need to add your GitHub Personal Token. (Note: The personal token does not need any permissons! Leave it all empty). Your Personal Token is saved in Local Storage in your browser and is only sent to github.com.
To create a Personal Token go to GitHub and then head over to settings. In the development settings you find Personal access tokens. Click Generate New Token.
